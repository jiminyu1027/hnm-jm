# H&M 쇼핑몰

리액트를 이용한 쇼핑몰 웹사이트.

- Demo : [https://jm-hnm.netlify.app/](https://jm-hnm.netlify.app/)

## 개발 목표

리액트 Redux, Redux-thunk, Router 사용해보기

## 사용 기술

- React
- React-router
- React-redux
- Redux-thunk
- Bootstrap

## 기능

- 유저는 메뉴와 상품들을 볼 수 있다.
- 유저는 로그인을 할 수 있다.
- 유저는 상품디테일을 보기 위해 로그인을 해야한다.
- 로그인한 유저는 상품디테일정보를 볼 수 있다.
- 유저는 상품을 검색할 수 있다.
- 유저는 로그아웃할 수 있다.

## 개선 사항

- 로그인을 했을 때, 로그아웃 버튼을 누르면 로그아웃이 되도록 하기
